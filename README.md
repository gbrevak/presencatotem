# Corban360 Totem Web

Primeira versao de uma aplicacao web para totem de stand com foco em branding animado, glassmorphism e navegacao simples para eventos.

## O que esta pronto

- Home responsiva com cinco menus clicaveis: Portfolio, Plataforma, Depoimentos, Tecnologias e VendeAI.
- Alternancia de tema com botao de sol e lua para leituras claro e escuro.
- Fundo dinamico desfocado com movimento suave.
- Modulo de depoimentos em formato de stories, com barra de progresso e avanco por toque.
- Abertura da plataforma externa em https://portal.soupresenca.com.br.

## Proximas etapas

- Inserir coordenadas e conteudos reais dos submenus.
- Popular Portfolio, Tecnologias e VendeAI.
- Adicionar modo ocioso com carrosseis e outros frames.

## Scripts

- `npm run dev` inicia o ambiente local.
- `npm run build` gera a build de producao.
- `npm run preview` abre a build localmente.
